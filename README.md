Github Wiki Search
==================

This is greasemonkey script.  
Search wiki of the repository on Github.

Browser
-------

- Firefox
- Chrome
- Safari  
  Note: need [NinjaKit](http://d.hatena.ne.jp/os0x/20100612/1276330696)

-------
